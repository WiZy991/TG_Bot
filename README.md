# Quiz Telegram Bot

Учебный проект телеграм-бота для проведения квиза.
Бот задаёт вопросы, принимает ответы пользователя, сохраняет последний результат и позволяет посмотреть статистику.

Проект выполнен в рамках учебного задания по работе с Telegram Bot API и библиотекой aiogram 3.


##  Возможности бота
- Прохождение квиза из 10 вопросов
- Выбор ответов с помощью кнопок
- Отображение выбранного ответа в чате
- Подсчёт результата
- Сохранение последнего результата пользователя
- Просмотр статистики


##  Команды бота
- `/start` — запуск бота и приветствие
- `/quiz` — начать прохождение квиза
- `/stats` — показать последний результат пользователя

##  Используемые технологии
- Python 3.9
- aiogram 3
- aiosqlite
- asyncio

##  Установка и запуск

1. Клонировать репозиторий:
git clone <ссылка_на_репозиторий>

Установить зависимости:
pip install -r requirements.txt

Запустить бота:
python bot.py
